    rdx-grid

    Since 1.0.0
    @mixin rdx-grid($auto, $center, $flush, $reverse, $stack, $vcenter) { ... }

    Description

    Grid Generates grid code for a containing element. Useful if users do not wish to use the CSS grid classes in their markup.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $auto

    Enables the auto-grid function where items will wrap

    		Bool—none
    $center

    Horizontally centers items

    		Bool—none
    $flush

    Removes all spacing between items

    		Bool—none
    $reverse

    Reveres the order of the items

    		Bool—none
    $stack

    Display items in a column rather than horizontally

    		Bool—none
    $vcenter

    Vertically center items in grid

    		Bool—none

    Output

    Outputs flexbox CSS properties for generating the Radix grid system

    Requires

    • [function] column-width

    Author

    • James Wigger
